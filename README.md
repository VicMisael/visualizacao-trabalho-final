# Visualização - Trabalho Final - Fortaleza em Dados: Um panorama socioeconômico


Este projeto foi desenvolvido utilizando **Angular 21**, **Vega-Lite**, **D3** e **Tailwind CSS** para a construção de visualizações interativas.

## Pré-requisitos

Antes de iniciar, certifique-se de possuir os seguintes softwares instalados:

- **Node.js** (versão 24 LTS ou superior recomendada)
- **npm** (versão 11.6.2 ou superior)

Verifique as versões instaladas:

```bash
node -v
npm -v
```

## Instalando as dependências

Instale todas as dependências do projeto utilizando:

```bash
npm install
```

## Executando o projeto

Para iniciar o servidor de desenvolvimento:

```bash
npm start
```

Após iniciar, acesse:

```
http://localhost:4200
```

O projeto será recompilado automaticamente sempre que houver alterações nos arquivos.


## Deploy (GitHub Pages)

O projeto possui configuração para deploy utilizando GitHub Pages.

Execute:

```bash
npm run deploy
```

## Tecnologias utilizadas

- Angular 21
- TypeScript 5
- Vega-Lite
- Vega Embed
- D3.js
- Tailwind CSS
